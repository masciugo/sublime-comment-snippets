# Sublime Text 2 Comment Snippets for ruby

A set of nice Sublime Text 2 comment snippets for ruby.

## Snippets

* `comt` - TODO
* `comf` - FIXME
* `comn` - NOTE
* `comb` - Nicely formatted block comment
* `comd` - Nice divider comment
